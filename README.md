## Bank Transactions CLI

Bank Transactions CLI — консольное приложение для обработки банковских транзакций из различных источников (JSON, CSV, XLSX) 
с возможностью фильтрации, сортировки, маскирования данных и конвертации валют.

Реализованы основы работы с файлами, логирование, декораторы, генераторы, работа с API и построение CLI-интерфейса.

### Возможности

**Программа:**

- отображает количество найденных операций

- считает количество операций по категориям

- выводит список отфильтрованных транзакций

#### Загрузка данных

- Поддерживаются форматы:

  - JSON

  - CSV

  - XLSX (Excel)

- Данные приводятся к единому формату для дальнейшей обработки.

#### Фильтрация транзакций

- По статусу (EXECUTED, CANCELED, PENDING)

- По дате (сортировка по возрастанию / убыванию)

- По валюте

- По ключевому слову в описании (regex)

#### Маскирование данных

- Маска банковской карты:
XXXX XX** **** XXXX

- Маска банковского счета:
**XXXX

#### Конвертация валют

- Поддерживается конвертация:

    - USD → RUB

    - EUR → RUB

- Через внешний API сервиса apilayer (Exchange Rates API).

- API-ключ передается через переменную окружения:
```
API_KEY=your_api_key_here
```

### Установка
```
git clone https://github.com/simbarilion/Client_Banking_Widget
cd bank_transactions
pip install -r requirements.txt
```
Для Poetry:
```
poetry install
poetry shell
```
**Запуск**
```
python main.py
```
После запуска отобразится интерактивное меню:
```
1. Получить информацию из JSON
2. Получить информацию из CSV
3. Получить информацию из XLSX
4. Конвертировать сумму в RUB
5. Выйти
```

Пример работы

1. Выбор источника данных
2. Фильтрация по статусу
3. Сортировка по дате
4. Дополнительные фильтры
5. Вывод итогового списка транзакций
